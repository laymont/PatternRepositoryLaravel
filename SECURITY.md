# Security Policy

## Versiones Soportadas

| Versión | Soportada          |
| ------- | ------------------ |
| 4.0.x   | :white_check_mark: |
| 3.0.x   | :white_check_mark: |
| < 3.0   | :x:                |

## Reportar una Vulnerabilidad

Si descubres una vulnerabilidad de seguridad, por favor:

1. **NO** crees un issue público
2. Envía un email al maintainer: laymont@example.com (o usa GitHub Security Advisories)
3. Include detalles de la vulnerabilidad
4. Espera confirmación antes de publicar

## Requisitos de Seguridad

- No almacenamos credenciales ni datos sensibles
- Usamos prácticas seguras de desarrollo
- Dependencies actualizadas regularmente
- Code review antes de merge

## Versión de Seguridad Actual

- Laravel: `^13.0`
- PHP: `>=8.3` (con soporte de seguridad activo)
